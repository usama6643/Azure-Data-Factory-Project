# Azure-Data-Factory-Project
Excited to share my latest Data Engineering project built using Azure Data Factory (ADF)!
I recently designed and deployed an end-to-end automated data pipeline that bridges on-premises data infrastructure with the Azure Cloud ecosystem.
Here is a breakdown of what I built and the core challenges I solved:
🔹 The Architecture:
Data Ingestion: Connected local file systems to the cloud using a Self-Hosted Integration Runtime (SHIR) for secure and efficient ingestion into Azure Data Lake Storage (ADLS Gen2) — Bronze Layer.
Data Transformation: Developed complex ADF Data Flows to clean, filter, and structure raw data.
Storage Optimization: Converted and optimized the transformed data into high-performance Delta Tables — Silver Layer
